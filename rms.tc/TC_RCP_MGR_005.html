TC_RCP_MGR_005

Recipe 검색 기능

Functional High
01

Precondition

  1. Recipe 10개 이상 존재, 이름이 다양함
  2. Recipe Manager 진입
02

Test Steps

  1. 좌측 트리 상단 검색창(🔍)에 "spinach" 입력
  2. 실시간 필터링 결과 확인 (RMSRecipeWspinach_gnocci 등 매칭)
  3. 검색어 삭제 → 전체 목록 복귀
  4. 대소문자 다르게 입력 ("SPINACH") → 동일 결과 확인
  5. 존재하지 않는 키워드 입력 → 빈 목록 및 안내 메시지 확인
03

Test Data

Search terms: - "spinach" → RMSRecipeWspinach_gnocci - "SDC" → SDC_A1WPP_GZ3_LENS_ADI - "xyz999" → 결과 없음
04

Expected Result

  1. 입력한 문자열이 Recipe 이름에 포함된 항목만 트리에 표시
  2. 대소문자 구분 없이 매칭
  3. 검색어 삭제 시 전체 목록 즉시 복귀
  4. 결과 없음 시 "검색 결과가 없습니다" 등의 안내 표시
  5. 검색 중 선택되어 있던 Recipe가 필터에서 제외되면 Recipe Information 영역 상태 유지 또는 Empty 전환 (정책에 따름)
05

비고 (QA · 개발 메모)

검색 대상 범위: Recipe 이름만인지, Description이나 Collected Equipment까지 포함인지 확인 필요.

디바운스: 타이핑 중 매 키마다 필터링하지 않고 150~300ms 디바운스 적용 권장.

06

제안 사항

UX 개선 검색 결과에서 매칭된 문자열을 하이라이트하면 가독성이 좋아집니다. "RMSRecipeWspinach_gnocci" 형태.

기획 검토 "고급 검색" 기능(예: Collected Equipment별, 수집일 범위) 필요성 검토. Recipe가 수백 개로 늘어날 경우 단순 텍스트 검색으론 한계가 있습니다.

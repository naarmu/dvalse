TC_SYS_LGN_003

빈 ID 또는 빈 PW 제출 시 검증

Negative High
01

Precondition

  1. Login 팝업 표시
02

Test Steps

  1. ID 빈칸, PW 입력 → Login 클릭
  2. ID 입력, PW 빈칸 → Login 클릭
  3. 모두 빈칸 → Login 클릭
03

Test Data

공백 입력 / 빈 문자열
04

Expected Result

  1. 빈 값 상태에서는 Login 버튼 비활성화 또는 클릭 시 에러
  2. "ID를 입력해주세요", "비밀번호를 입력해주세요" 같은 안내
  3. 서버로 요청이 전송되지 않음 (클라이언트 차단)
05

비고 (QA · 개발 메모)

06

제안 사항

UX 개선 필수 필드 미입력 시 Login 버튼을 비활성화 유지하는 방식을 권장드립니다. 불필요한 에러 팝업을 줄일 수 있습니다.

TC_EQP_GRP_007

그룹 트리 계층 표시 일관성

UI Medium
01

Precondition

  1. 최소 3개의 그룹, 각 그룹당 2대 이상 장비가 존재
  2. Edit Equipment Groups, Equipment Manager, Dashboard 화면 모두 접근 가능
02

Test Steps

  1. Edit Equipment Groups의 그룹 목록 순서와 장비 표시 확인
  2. Equipment Manager의 좌측 트리와 위 화면의 그룹 순서·내용 비교
  3. Dashboard의 "By Equipment Groups" 필터 순서와 비교
  4. Deploy Recipes 화면의 장비 선택 영역에서의 그룹 표시 확인
  5. 세 화면에서의 그룹 순서·소속 장비 목록이 일치하는지 확인
03

Test Data

GROUP_A (2대), GROUP_B (3대), GROUP_C (1대), 미지정 (1대)
04

Expected Result

  1. 그룹 순서가 모든 화면에서 동일함 (알파벳순 또는 생성순 등 일관된 기준)
  2. 소속 장비 목록이 각 화면에서 일치함
  3. 그룹 노드의 펼침/접힘 UI 스타일이 화면 간 통일되어 있음
  4. 아이콘, 색상, 폰트 등 시각적 요소도 일관됨
  5. "미지정" 노드가 항상 같은 위치(맨 위 또는 맨 아래)에 표시됨
05

비고 (QA · 개발 메모)

정렬 기준: 알파벳순 / 생성순 / 사용자 정의 순 중 어떤 기준을 사용하는지 명시 필요. 세 화면 모두 같은 기준이어야 함.

06

제안 사항

정보 구조 "미지정" 그룹의 표시 위치는 항상 맨 아래를 권장드립니다. 일반 그룹 위에 섞이면 사용자가 혼란을 겪을 수 있습니다.

디자인 시스템 그룹/장비 아이콘, 상태 컬러 등을 공통 컴포넌트로 관리하여 세 화면 모두 동일한 컴포넌트를 재사용하도록 하는 것이 유지보수에 유리합니다.

TC_EQP_REG_001

신규 장비 등록 (정상 케이스)

Functional Critical
01

Precondition

  1. Vendor 또는 Admin 계정으로 로그인됨
  2. 서버에 존재하지 않는 Equipment ID (예: EQ_NEW_01)를 준비
  3. 해당 장비 PC에는 향후 Agent를 설치할 예정이지만 아직 설치되지 않음
02

Test Steps

  1. Equipment → Register Equipment 화면 진입
  2. Equipment ID: EQ_NEW_01 입력
  3. Equipment Name, IP, Group 등 필수 필드 입력
  4. Description (선택) 입력
  5. Save 또는 Register 버튼 클릭
  6. Equipment Manager로 이동하여 등록된 장비 확인
03

Test Data

Equipment ID: EQ_NEW_01 Equipment Name: Test New Equipment IP Address: 192.168.1.150 Group: GROUP_A Description: (optional) QA 등록 테스트용
04

Expected Result

  1. 등록 성공 메시지 표시 (Toast 또는 팝업)
  2. Equipment Manager 목록에 EQ_NEW_01이 추가됨
  3. 초기 연결 상태는 Offline으로 표시됨 (Agent 미설치)
  4. 등록 이력이 RMS Log 또는 Server Log에 기록됨 (등록자, 시각 포함)
  5. 등록된 장비를 선택하면 우측 상세에 입력한 정보가 모두 정확히 표시됨
05

비고 (QA · 개발 메모)

중복 여부 확인: 본 TC는 정상 케이스이므로 Equipment ID가 확실히 신규임을 사전 확인 필요.

등록 후 전환: 등록 완료 시 자동으로 Equipment Manager 화면으로 이동하는지, 현재 화면에 머무는지 UX 동작 확인.

06

제안 사항

UX 개선 등록 성공 후 바로 "이 장비에 Agent 설치하기" 안내 또는 Agent Installer 다운로드 링크를 제공하면, 등록 → 연결까지의 플로우가 매끄러워집니다.

기획 검토 등록 폼에 들어가는 필수 필드와 선택 필드의 구분이 명확해야 합니다. 현재 가정: Equipment ID, Name, IP는 필수. 확정 시 알려주세요.

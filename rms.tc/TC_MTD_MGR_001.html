TC_MTD_MGR_001

Method Manager 진입 및 목록 로딩

Functional Critical
01

Precondition

  1. 서버에 Method .psm 파일 다수 등록됨
  2. 로그인 상태
02

Test Steps

  1. Method 메뉴 → Method Manager 진입
  2. 좌측 Method 트리, 중앙/우측 상세 영역 확인
  3. 버전 노드 펼침 확인
03

Test Data

Methods: 10개 이상, 각 1~3 버전
04

Expected Result

  1. 화면 5초 이내 로딩
  2. 좌측 트리에 모든 Method와 하위 버전 표시
  3. Recipe Manager와 동일한 레이아웃 패턴
05

비고 (QA · 개발 메모)

Recipe와 동일 패턴: Method는 .psm 확장자를 쓰지만 관리 UI·워크플로우는 Recipe와 동일하게 가정.

06

제안 사항

일관성 Method와 Recipe는 UI 구조를 최대한 동일하게 유지해 학습 비용을 낮추는 것이 좋습니다.

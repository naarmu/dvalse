TC_RCP_DIR_001

로컬 PC에서 Recipe(.psr) 파일 서버로 직접 업로드

Functional Critical
01

Precondition

  1. 로컬 PC에 유효한 .psr 파일(test_recipe.psr) 준비
  2. Recipe Manager 또는 Direct Upload 기능 진입 가능
02

Test Steps

  1. Recipe Manager에서 "Upload" 버튼 클릭 (slide-27 참고)
  2. 로컬 PC의 test_recipe.psr 선택
  3. 업로드 진행
  4. 서버 Recipe 목록에 등록되는지 확인
03

Test Data

File: test_recipe.psr (로컬 PC)
04

Expected Result

  1. .psr 파일이 서버로 업로드됨
  2. Recipe Manager 목록에 신규 Recipe로 표시
  3. 파라미터가 정상 파싱되어 Contents 영역에 표시
  4. Description 또는 출처 필드에 "Direct Upload" 표시
  5. 이력 기록
05

비고 (QA · 개발 메모)

주요 특징: 장비를 거치지 않고 로컬 → 서버 직접 전송하는 경로. 기존 "장비 → 서버 수집"과 다른 플로우.

06

제안 사항

정보 구조 Direct Upload한 Recipe는 일반 수집 Recipe와 시각적으로 구분되어야 합니다. 출처가 다르면 검증 책임 주체도 다르기 때문입니다.

TC_SYS_PND_001

Pending Recipes 팝업 자동 표시 조건

Functional Critical
01

Precondition

  1. 장비에서 Recipe가 새로 생성 · 수정되어 서버 미등록 상태가 존재
02

Test Steps

  1. 로그인 후 메인 진입
  2. Pending Recipes 팝업이 자동 표시되는지 확인
  3. 팝업에 표시된 대기 항목 수 확인
03

Test Data

Pending Recipes: 3개 (사전 세팅)
04

Expected Result

  1. 로그인 직후 Pending Recipes 팝업 자동 표시
  2. 대기 항목 수와 목록 표시
  3. Pending이 0건이면 팝업 미표시
05

비고 (QA · 개발 메모)

팝업 조건: Pending 0건일 때는 팝업을 띄우지 않는 것이 UX에 좋음.

06

제안 사항

UX 개선 Pending 항목이 많을 경우 매번 로그인 시 팝업이 떠서 번거로울 수 있습니다. "오늘 하루 보지 않기" 옵션을 제공하면 편리합니다.

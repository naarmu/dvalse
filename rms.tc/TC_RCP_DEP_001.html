TC_RCP_DEP_001

Deploy Recipes 화면 진입 및 Recipe · 버전 선택

Functional Critical
01

Precondition

  1. 배포 가능한 Recipe 및 Master Recipe 지정된 버전이 존재
  2. Online 장비 2대 이상 연결됨
02

Test Steps

  1. Recipe 메뉴 → Deploy Recipe 진입
  2. 좌측 Recipe 목록에서 배포할 Recipe 선택
  3. Available Versions 목록에서 특정 버전 선택
  4. 선택된 Recipe/버전 정보가 상단 요약에 표시되는지 확인
03

Test Data

Recipe: 213 Selected Version: v2 (Master)
04

Expected Result

  1. 화면 3초 이내 진입, Recipe 목록 표시
  2. Recipe 선택 시 Available Versions 목록이 우측에 표시
  3. Master 버전은 별도 배지 또는 아이콘으로 강조
  4. 버전 선택 시 선택된 Recipe 버전 정보 요약 표시 (slide-30, slide-53 참고)
05

비고 (QA · 개발 메모)

RMS UI 참고: slide-30(Deploy Recipe)에서 "Master Version" 체크박스와 "Available Versions" 테이블 구조 확인.

06

제안 사항

UX 개선 Master 버전을 기본 선택값으로 제공하는 것을 권장드립니다. 대부분의 배포는 Master 대상으로 이뤄지므로 기본값이 유용합니다.

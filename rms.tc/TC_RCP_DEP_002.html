TC_RCP_DEP_002

배포 대상 장비 단일 선택 후 배포

Functional Critical
01

Precondition

  1. Deploy Recipes에서 Recipe 213 v2가 선택된 상태
  2. 장비 EQ_001이 Online 상태로 연결됨
02

Test Steps

  1. 배포 대상 장비 영역에서 EQ_001 하나만 선택
  2. 배포(Deploy) 버튼 클릭
  3. 확인 팝업에서 대상 장비 및 Recipe 정보 확인 후 "확인"
  4. 배포 진행 상태 표시 확인
  5. 배포 완료 후 EQ_001의 Deployed Recipe 정보 갱신 확인
03

Test Data

Target: EQ_001 Recipe: 213 v2
04

Expected Result

  1. 대상 장비 단일 선택 정상 동작
  2. Deploy 전 확인 팝업에 장비 ID, Recipe, 버전 명시
  3. 배포 중 진행률 또는 상태 인디케이터 표시
  4. 성공 시 Toast/팝업 알림, EQ_001의 Deployed Recipe ID/Version 업데이트
  5. Deployment Log에 이력 기록 (배포자, 시각, Recipe, 버전, 장비, 결과)
05

비고 (QA · 개발 메모)

원자성: 단일 장비 배포는 atomic해야 함. 부분 성공 케이스 없음.

06

제안 사항

UX 개선 배포 확인 팝업에 "현재 배포된 Recipe → 새 Recipe" 형태로 변경 내용을 명시하면, 실수 배포를 예방할 수 있습니다.

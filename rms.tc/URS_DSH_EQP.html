URS_DSH_EQP

장비 운영 대시보드

User Requirement Overview
01

기능 개요

장비 운영 현황에 특화된 대시보드이다. 사용자는 이 화면에서 각 장비의 가동 상태, 현재 사용 중인 Recipe, 알람 발생 여부 등을 한눈에 확인한다.

RMS Dashboard가 시스템 전반의 요약을 보여준다면, Equipment Dashboard는 장비 단위의 운영 현황에 집중한다. 장비 엔지니어가 일상적으로 머무는 화면이며, 이상 장비를 빠르게 발견하기 위한 도구이다.

02

사용자 니즈

주요 사용자

  • 장비 엔지니어 — 담당 장비의 가동 현황 점검
  • 공정 엔지니어 — 작업 대상 장비의 현재 Recipe 확인

Pain Point (RMS 도입 전)

  • 장비별 가동 현황을 보려면 각 장비를 직접 방문해야 함
  • 여러 장비의 상태를 동시에 비교하기 어려움
  • 알람 발생을 사후에야 인지

기대 효과

  • 전체 장비의 가동 상태를 단일 화면에서 조감
  • 장비별 사용 Recipe 즉시 확인 가능
  • 이상 장비를 시각적으로 빠르게 발견
03

기능 요구사항

■ FR-1. 장비 목록 표시

  • FR-1.1 Dashboard 진입 시 RMS에 등록된 모든 장비가 카드 또는 그리드 형태로 표시된다.
  • FR-1.2 각 장비 카드에는 장비명, 연결 상태, 현재 사용 Recipe, 알람 표시가 포함된다.
  • FR-1.3 사용자는 그룹별 또는 상태별로 필터링하여 표시할 수 있어야 한다.

■ FR-2. 장비 상세 정보

  • FR-2.1 장비 카드를 선택하면 상세 영역에 가동 현황, 사용 중인 Recipe·Method 목록, 최근 배포 이력이 표시된다.
  • FR-2.2 상세 영역의 Recipe·Method 항목을 클릭하면 해당 자료의 상세 화면으로 이동한다.

■ FR-3. 상태 표시

  • FR-3.1 각 장비의 상태(Online / Offline / Alarm)가 시각적으로 구분 표시되어야 한다.
  • FR-3.2 알람 발생 장비는 별도 색상·아이콘으로 강조 표시한다.
  • FR-3.3 알람 카드에는 알람 사유와 발생 시각이 함께 표시된다.

■ FR-4. 실시간 갱신

  • FR-4.1 장비 상태 변경(Online ↔ Offline, Alarm 발생/해제)은 일정 주기로 화면에 자동 반영된다.
  • FR-4.2 사용자는 수동 새로고침으로 즉시 갱신할 수 있다.
  • FR-4.3 마지막 갱신 시각이 화면에 표시된다.

■ FR-5. Empty 상태 처리

  • FR-5.1 등록된 장비가 0개인 경우 안내와 다음 액션 유도(Register Equipment 바로가기)가 표시된다.
04

비기능 요구사항

■ NFR-1. 성능

  • NFR-1.1 장비 100대 이하 환경에서 Dashboard 로딩은 5초 이내 완료되어야 한다.
  • NFR-1.2 장비 수가 많은 환경에서는 가상 스크롤 또는 페이지네이션을 적용한다.

■ NFR-2. 권한

  • NFR-2.1 Equipment Dashboard는 모든 사용자(All Roles)가 조회 가능하다.

■ NFR-3. 가시성

  • NFR-3.1 상태 표시는 색상 + 아이콘 병행으로 한다.
  • NFR-3.2 알람 발생 장비는 화면에서 즉시 눈에 띄도록 강조 표시한다.
05

제약사항 · 정책

  • 정책 Dashboard 자동 갱신 주기는 30초이다. (RMS Dashboard와 동일)
  • 정책 알람 발생 장비는 별도 시각 강조로 표시되어야 한다.
  • 정책 장비 카드의 Recipe·Method 항목 클릭 시 해당 상세 화면으로 자동 이동한다.
  • 제약 장비 수가 많은 환경에서는 가상 스크롤·페이지네이션을 적용해 초기 로딩 부담을 줄인다.
  • 제약 알람 정의·발생 로직은 장비 Agent 또는 별도 모니터링 시스템에서 관리하며, RMS는 표시만 담당한다.
06

관련 Test Cases

이 URS의 요구사항을 검증하는 QA 테스트 케이스 5건:

  • TC_DSH_EQP_001 — Equipment Dashboard 장비 목록 로딩 (FR-1, NFR-1.1)
  • TC_DSH_EQP_002 — 장비 선택 시 가동 현황 · 사용 Recipe 표시 (FR-2)
  • TC_DSH_EQP_003 — 장비 상태 (온라인 · 오프라인 · 알람) 표시 정확성 (FR-3, NFR-3.1)
  • TC_DSH_EQP_004 — 장비 상태 변경 시 실시간 갱신 (FR-4)
  • TC_DSH_EQP_005 — 장비 없음 상태 표시 (FR-5)

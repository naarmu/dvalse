TC_SYS_LGN_007

비활성(Disabled) 계정 로그인 차단

Negative High
01

Precondition

  1. Settings에서 특정 계정을 Disabled 상태로 설정
02

Test Steps

  1. Disabled 계정의 ID/PW로 로그인 시도
  2. 에러 메시지 확인
03

Test Data

Account: disabled_user (Disabled 상태)
04

Expected Result

  1. 로그인 차단
  2. "계정이 비활성화되어 있습니다" 또는 모호한 인증 실패 메시지 (보안 정책에 따라)
  3. 시도 이력 기록
05

비고 (QA · 개발 메모)

메시지 정책: "비활성 계정"임을 명시하면 계정 존재를 확인시켜주는 보안 이슈. 일반 실패 메시지가 보안상 안전.

06

제안 사항

보안 계정 비활성 안내는 관리자에게만 명시하고, 로그인 화면에서는 일반 실패 메시지를 표시하는 것이 보안상 안전합니다.

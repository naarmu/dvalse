TC_EQP_MGR_006

장비 그룹 트리 구조 표시

UI High
01

Precondition

  1. 복수 그룹(GROUP_A, GROUP_B, "미지정")이 존재함
  2. 각 그룹에 장비가 1대 이상 소속됨
  3. 그룹에 속하지 않은 장비도 1대 이상 존재
02

Test Steps

  1. Equipment Manager 화면 진입
  2. 좌측 트리에 그룹이 부모 노드, 장비가 자식 노드로 표시되는지 확인
  3. 그룹 노드의 펼침/접힘 토글 동작 확인
  4. 그룹에 속하지 않은 장비가 어떻게 표시되는지 확인
  5. 장비가 없는 빈 그룹의 표시 방식 확인
03

Test Data

GROUP_A: EQ_001, EQ_002 GROUP_B: EQ_003 미지정: EQ_004 빈 그룹 GROUP_C: (소속 장비 없음)
04

Expected Result

  1. 그룹이 부모 노드, 장비가 자식 노드인 2 depth 트리 구조로 표시
  2. 그룹 노드 좌측에 펼침/접힘 아이콘(▶/▼)이 명확히 표시됨
  3. 소속 없는 장비는 "미지정" 또는 "Ungrouped" 노드 하위에 표시
  4. 빈 그룹은 펼쳐도 자식이 없음을 알 수 있는 상태로 표시 (또는 숨김 정책에 따름)
  5. 각 그룹 노드 옆에 소속 장비 수가 표시됨 (예: GROUP_A (2))
05

비고 (QA · 개발 메모)

트리 아이콘: 그룹/장비 구분을 위한 아이콘이 존재하는지, 의미가 명확한지 확인.

Deep Tree: 현재는 2 depth 구조로 가정했지만, 향후 서브그룹 요구가 생길 수 있으므로 확장성 고려 필요.

06

제안 사항

정보 구조 그룹명 옆에 소속 장비 수 표시 (예: "GROUP_A (3)") 를 강력히 권장드립니다. 그룹을 펼치지 않고도 규모를 빠르게 파악할 수 있어 UX가 크게 향상됩니다.

UX 개선 빈 그룹은 일반 그룹과 시각적으로 약간 구분(텍스트 불투명도 60% 등)하여 사용자가 "쓰지 않는 그룹"을 인지하기 쉽게 할 수 있습니다.

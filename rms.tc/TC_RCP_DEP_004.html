TC_RCP_DEP_004

장비 그룹 단위 배포

Functional High
01

Precondition

  1. 그룹 "9F [DRD]"에 장비 6대 소속
  2. 이 중 일부는 Online, 일부는 Offline 상태
02

Test Steps

  1. 대상 선택에서 "그룹 단위 선택" 모드로 전환 (기능 존재 시)
  2. "9F [DRD]" 그룹 선택 시 소속 장비 자동 선택 확인
  3. Offline 장비의 처리 방식 확인 (자동 제외 or 경고)
  4. Deploy 실행 후 결과 확인
03

Test Data

Group: 9F [DRD] Members: EQ_001(Online), EQ_002(Online), ..., EQ_006(Offline)
04

Expected Result

  1. 그룹 선택 시 소속 장비 일괄 선택
  2. Offline 장비는 선택에서 자동 제외 또는 경고 표시 후 사용자 결정
  3. 배포는 Online 장비만 진행
  4. Offline 장비는 "배포 대기" 또는 "스킵" 상태로 Deployment Log 기록
05

비고 (QA · 개발 메모)

Offline 장비 정책: "Offline 장비 재연결 시 자동 배포" 같은 재처리 로직이 있는지 정책 확인.

06

제안 사항

정책 확정 Offline 장비 배포 실패 처리 정책 확정이 필요합니다. 권장: "Offline 장비는 배포 대기열에 등록되었다가, 재연결 시 자동 배포 시도". 단, 자동 배포 전 관리자 승인 필요 여부는 별도 고려.

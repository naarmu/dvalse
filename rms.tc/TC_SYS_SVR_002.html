TC_SYS_SVR_002

잘못된 IP · 포트 입력 시 연결 실패 처리

Negative High
01

Precondition

  1. Server Setup 팝업 표시
02

Test Steps

  1. 존재하지 않는 IP 또는 잘못된 포트 입력
  2. 연결 시도
  3. 에러 처리 확인
03

Test Data

Invalid IP: 192.168.255.255 Invalid Port: 99999
04

Expected Result

  1. 타임아웃 후 연결 실패 메시지 표시
  2. 입력 값 유지 (사용자가 수정 가능)
  3. Login 화면으로 전환되지 않음
05

비고 (QA · 개발 메모)

포트 범위: 유효 포트(1~65535) 범위 검증 필요.

06

제안 사항

UX 개선 IP와 포트는 실시간 형식 검증을 통해 입력 중에 오류를 알려주는 것이 좋습니다.

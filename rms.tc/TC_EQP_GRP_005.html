TC_EQP_GRP_005

빈 그룹 삭제

Functional Medium
01

Precondition

  1. 소속 장비가 없는 빈 그룹 GROUP_EMPTY가 존재함
02

Test Steps

  1. Edit Equipment Groups에서 GROUP_EMPTY 선택
  2. Delete Group (휴지통 아이콘) 클릭
  3. 확인 팝업 내용 확인 (소속 장비 없음이 언급되는지)
  4. "확인" 선택
  5. 그룹 목록에서 GROUP_EMPTY 제거 확인
  6. Dashboard, Equipment Manager에서도 해당 그룹 사라짐 확인
03

Test Data

Group: GROUP_EMPTY (Members: 없음)
04

Expected Result

  1. 삭제 확인 팝업이 표시됨 (빈 그룹의 경우 간단한 확인)
  2. "확인" 시 그룹이 즉시 삭제됨
  3. Dashboard 필터, Equipment Manager 트리에서 사라짐
  4. 삭제 이력이 Server Log에 기록됨
  5. "취소" 시 그룹 유지됨
05

비고 (QA · 개발 메모)

팝업 톤: 빈 그룹 삭제와 소속 장비가 있는 그룹 삭제(TC_EQP_GRP_006)는 팝업 메시지 톤이 달라야 함. 빈 그룹은 "정말 삭제하시겠습니까?" 정도면 충분.

06

제안 사항

UX 개선 빈 그룹 삭제는 경량 확인(간단한 confirm)으로, 소속 장비가 있는 그룹 삭제는 경고형 모달로 UX를 차별화하는 것을 권장드립니다. 둘 다 같은 팝업이면 사용자가 경고를 무시하는 습관이 생길 수 있습니다.

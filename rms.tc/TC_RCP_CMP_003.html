TC_RCP_CMP_003

불일치 파라미터 하이라이트 (색상) 표시

UI Critical
01

Precondition

  1. 값이 다른 파라미터를 10개 이상 가진 2개 Recipe
  2. Compare Recipes에서 두 Recipe가 비교된 상태
02

Test Steps

  1. 비교 결과 테이블에서 하이라이트된 행들 확인
  2. 하이라이트 색상의 명도와 대비가 적절한지 확인
  3. 다크모드가 있다면 다크모드에서도 가독성 확인
  4. 색각 이상자를 위한 패턴/아이콘 병행 여부 확인
03

Test Data

불일치 파라미터 10+ 개 존재
04

Expected Result

  1. 불일치 행이 명확한 배경색으로 강조됨
  2. 일치 행과의 시각적 대비가 확실함
  3. 하이라이트 색상이 너무 강하지 않아 가독성 유지
  4. 색상 외에도 아이콘(⚠️ 또는 ●) 같은 보조 표시가 있으면 더 좋음
05

비고 (QA · 개발 메모)

접근성: Park Systems RMS가 반도체 공정 환경에서 사용됨을 고려하면, 색각이상자 접근성도 검토 필요. 색상만으로 의미 전달하지 않도록.

slide-53 참고: Phase 1 주요기능 [4]에서 "일치하지 않는 값 강조 표시"로 이미 구현된 기능.

06

제안 사항

접근성 하이라이트는 색상 + 아이콘 병행을 권장드립니다. 색각이상자(인구의 약 8%)도 동일한 정보를 인지할 수 있도록.

UX 개선 하이라이트된 행을 클릭하면 해당 파라미터의 상세 비교(이전 값 vs 새 값)를 측면 패널에 표시하는 기능이 있으면 좋겠습니다.

TC_RCP_PRM_004

숨김 설정이 Recipe Manager 상세에 반영

Functional High
01

Precondition

  1. "Tip ID"가 Hide 설정됨
02

Test Steps

  1. Recipe Manager에서 Recipe 선택
  2. 우측 Recipe Contents 영역에서 "Tip ID"가 표시되지 않는지 확인
03

Test Data

Hidden: Tip ID
04

Expected Result

  1. Recipe Manager Contents 영역에 Hide 파라미터 미표시
  2. 설정 변경 시 즉시 반영
05

비고 (QA · 개발 메모)

모든 파라미터 표시 영역에 Hide 설정이 일관되게 적용되어야 함.

06

제안 사항

일관성 Hide 설정은 Recipe Manager, Compare, Deploy 등 모든 파라미터 표시 UI에 일관되게 적용되어야 합니다. 일부 화면에만 반영되면 사용자가 혼란을 겪습니다.

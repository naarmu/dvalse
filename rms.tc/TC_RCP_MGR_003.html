TC_RCP_MGR_003

장비 트리에서 장비 선택 시 해당 Recipe 필터링

Functional High
01

Precondition

  1. 장비 EQ_001에서 수집된 Recipe 3개, 장비 EQ_002에서 수집된 Recipe 2개가 각각 등록됨
  2. Recipe Manager 진입
02

Test Steps

  1. (화면에 장비 선택 또는 필터 기능이 있을 경우) 장비 필터에서 EQ_001 선택
  2. 좌측 Recipe 트리에 EQ_001의 Recipe만 표시되는지 확인
  3. 필터를 EQ_002로 변경 → 해당 장비의 Recipe만 표시 확인
  4. 필터 해제 → 전체 Recipe 표시 확인
03

Test Data

EQ_001: Recipe 213, Simulation, Walkthrough WLI EQ_002: Recipe SDC_A1WPP_GZ3_LENS_ADI, spinach_gnocci
04

Expected Result

  1. 장비 필터 적용 시 해당 장비에서 수집된 Recipe만 트리에 표시
  2. 다른 장비의 Recipe는 숨김 처리
  3. 필터 적용 중 선택된 Recipe가 필터 대상에서 제외되면 선택 해제 및 안내
  4. 필터 해제 시 전체 Recipe 복귀
  5. 필터 상태가 UI상 명확히 표시 (예: "Filter: EQ_001" 배지)
05

비고 (QA · 개발 메모)

기능 존재 여부 확인: Recipe Manager에 "장비별 필터" 기능이 구현되어 있는지 기획 확정 필요. 현재 slide-27 화면에는 상단 검색창만 보이고 장비 필터는 명시되지 않음.

다중 장비 수집 Recipe: 같은 Recipe가 여러 장비에서 수집된 경우의 표시 방식 확인.

06

제안 사항

기획 검토 Recipe Manager에서 장비별 필터가 필요한 시나리오가 실제로 있는지 확인이 필요합니다. 필요하다면 현재 검색창 옆에 장비 드롭다운 필터를 추가하는 것을 제안드립니다.

정보 구조 Recipe가 어느 장비에서 수집되었는지는 Recipe Information의 "Collected Equipment"에서 확인 가능하지만, 트리에서도 장비 태그나 아이콘으로 표시하면 시각적으로 파악이 쉽습니다.

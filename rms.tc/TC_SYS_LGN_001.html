TC_SYS_LGN_001

유효한 계정으로 로그인 성공

Functional Critical
01

Precondition

  1. RMS Client 설치 및 실행
  2. 서버에 유효한 계정(ID/PW) 등록됨
02

Test Steps

  1. Login 팝업이 자동 표시됨 (slide-18 참고)
  2. 유효한 ID/PW 입력
  3. Login 버튼 클릭
  4. 로그인 성공 후 기본 화면 진입 확인
03

Test Data

Account: admin_test / valid_password
04

Expected Result

  1. 인증 성공 후 메인 대시보드(또는 Role별 초기 화면)로 이동
  2. 상단 네비에 사용자명과 Role 표시 (RMS UI와 같이 "Ted Gim / Vendor")
  3. 로그인 이력 기록
05

비고 (QA · 개발 메모)

UI 참고: slide-18의 Login popup 형태와 일치해야 함.

06

제안 사항

UX 개선 "Remember Me" 체크박스를 제공하되, 보안을 위해 ID만 저장하고 PW는 저장하지 않는 정책을 권장드립니다.

TC_SYS_SVR_001

Server Setup 팝업에서 서버 주소 입력 및 저장

Functional Critical
01

Precondition

  1. RMS Client 최초 실행 또는 서버 정보가 없는 상태
02

Test Steps

  1. Server Setup 팝업 자동 표시 (slide-19 참고)
  2. Server IP 및 Port 입력
  3. 저장 또는 연결 시도
  4. Login 팝업으로 자동 전환 확인
03

Test Data

Server IP: 192.168.1.10 / Port: 8080
04

Expected Result

  1. 입력한 서버 정보가 로컬에 저장됨 (다음 실행 시 자동 적용)
  2. 서버 연결 성공 시 Login 화면으로 전환
  3. 실패 시 에러 메시지 및 재입력 유도
05

비고 (QA · 개발 메모)

UI 참고: slide-19 Server Setup popup.

06

제안 사항

UX 개선 "연결 테스트" 버튼을 제공하여, 저장 전에 실제 연결 가능 여부를 확인할 수 있게 해주세요.

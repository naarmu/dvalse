TC_SYS_LGN_004

서버 미연결 상태에서 로그인 시도

Negative High
01

Precondition

  1. RMS Client 실행 상태
  2. 서버가 꺼져 있거나 네트워크 단절
02

Test Steps

  1. 유효한 ID/PW 입력 후 Login 클릭
  2. 응답 대기
  3. 에러 처리 확인
03

Test Data

Server: Offline / Network: Disconnected
04

Expected Result

  1. 타임아웃 내(예: 10초) 서버 연결 실패 감지
  2. "서버에 연결할 수 없습니다" 안내 표시
  3. Server Setup 화면으로 유도하거나 재시도 옵션 제공
  4. Client가 crash하지 않음
05

비고 (QA · 개발 메모)

타임아웃 값: 너무 길면 UX 저하, 너무 짧으면 정상 상황에서도 실패 — 10초 권장.

06

제안 사항

UX 개선 서버 연결 실패 시 "Server Setup으로 이동" 버튼을 제공하면 사용자가 다음 액션을 바로 수행할 수 있습니다.

TC_RCP_VER_001

Recipe 버전 목록 조회

Functional Critical
01

Precondition

  1. Recipe "213"이 v1, v2, v3 등 다수 버전 보유
  2. Manage Recipe Versions 화면 접근 가능
02

Test Steps

  1. Manage Recipe Versions 진입
  2. Recipe 목록에서 "213" 선택
  3. 해당 Recipe의 모든 버전이 표시되는지 확인
  4. 각 버전의 주요 속성(Master 여부, 수집일, 코멘트) 확인
03

Test Data

Recipe: 213 - v3 (Master) - v2 - v1
04

Expected Result

  1. 선택된 Recipe의 모든 버전이 목록 표시
  2. Master 버전이 시각적으로 구분 (아이콘, 배지, 색상)
  3. 버전별 수집/등록 일시, 코멘트, 상태 표시
  4. slide-31(Recipe Versions) 구조 참고
05

비고 (QA · 개발 메모)

UI 참고: slide-31에 "Recipe Description", "Modifier", "Date", "Description" 컬럼 확인.

06

제안 사항

정보 구조 버전 목록에 "현재 배포된 장비 수" 컬럼을 추가하면, 해당 버전이 실제로 운영 중인지 빠르게 파악 가능합니다.

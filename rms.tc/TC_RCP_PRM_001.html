TC_RCP_PRM_001

파라미터 전체 목록 표시

Functional High
01

Precondition

  1. Recipe에 사용되는 파라미터가 여러 개 수집되어 서버 DB에 등록됨
02

Test Steps

  1. Manage Recipe Parameters 진입
  2. 수집된 모든 파라미터가 목록에 표시되는지 확인
  3. 각 파라미터의 속성(이름, 타입, 단위, 표시 여부) 확인
03

Test Data

파라미터 예시: Size, Thickness, Die Based, Die X, Die Y, Measure Points, Tip ID, Tip Qualification, ...
04

Expected Result

  1. 파라미터 전체 목록 표시
  2. 각 항목에 이름, 데이터 타입, 단위, 현재 표시 상태(Show/Hide) 포함
  3. 정렬/검색 가능
  4. slide-32(Manage Recipe Parameters) 구조 참고
05

비고 (QA · 개발 메모)

파라미터 갱신: 새로운 Recipe가 수집될 때 기존에 없던 파라미터가 발견되면 목록에 자동 추가되는지 확인.

06

제안 사항

정보 구조 파라미터를 카테고리별로 그룹화(예: 측정, 정렬, Tip, Die)하면 수십 개 파라미터를 관리하기 쉽습니다.

TC_EQP_GRP_003

그룹 이름 변경

Functional Medium
01

Precondition

  1. 기존 그룹 GROUP_OLD가 존재함 (소속 장비 2대 포함)
02

Test Steps

  1. Edit Equipment Groups에서 GROUP_OLD 선택
  2. 우측 편집 영역에서 Group Name을 GROUP_RENAMED로 변경
  3. Update 버튼 클릭
  4. 그룹 목록에 반영되는지 확인
  5. Equipment Manager 트리, Dashboard 필터 등 다른 화면에서도 갱신되는지 확인
03

Test Data

Before: GROUP_OLD (Members: EQ_001, EQ_002) After: GROUP_RENAMED
04

Expected Result

  1. 그룹명이 GROUP_RENAMED로 변경되어 목록에 표시
  2. 소속 장비의 그룹 속성이 자동으로 새 이름으로 업데이트됨
  3. Equipment Manager 트리의 그룹 노드 이름이 갱신됨
  4. Dashboard의 "By Equipment Groups" 필터명도 갱신됨
  5. 변경 이력이 Server Log에 기록됨 (이전 이름, 새 이름, 변경자, 시각)
  6. 다른 그룹의 이름과 중복되지 않음을 사전 검증
05

비고 (QA · 개발 메모)

동시성: 두 사용자가 동시에 같은 그룹의 이름을 다르게 수정하는 경우 어떻게 처리되는지 확인 필요(마지막 쓰기 승리 vs. 충돌 감지).

06

제안 사항

UX 개선 그룹명 변경 전후의 값이 동시에 보이는 확인 모달(Before: "GROUP_OLD" → After: "GROUP_RENAMED")을 띄우면 실수를 줄일 수 있습니다.

데이터 무결성 그룹 이름 변경은 History 관점에서 중요한 이벤트입니다. 변경 이력에 이전 이름도 함께 기록되어 나중에 추적 가능해야 합니다.

TC_DSH_EQP_003

장비 상태(온라인 · 오프라인 · 알람) 표시 정확성

Functional Critical
01

Precondition

  1. 다양한 상태의 장비 존재 (Online, Offline, 알람)
02

Test Steps

  1. Equipment Dashboard에서 각 장비 카드 상태 확인
  2. Online/Offline 아이콘 및 알람 표시 확인
  3. 알람 발생 장비의 알람 아이콘 클릭 → 상세 내용 확인
03

Test Data

EQ_001: Online, 알람 있음 EQ_002: Offline EQ_003: Online, 정상
04

Expected Result

  1. 각 상태별로 구분되는 아이콘/색상
  2. 알람 있는 장비는 경고 아이콘(⚠️) 표시
  3. 알람 클릭 시 상세 내용 팝업 또는 사이드 패널
05

비고 (QA · 개발 메모)

UI 참고: slide-22 장비 카드 우측에 느낌표(⚠️) 아이콘이 일부 표시됨 — 알람 상태.

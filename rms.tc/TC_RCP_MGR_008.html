TC_RCP_MGR_008

Empty 상태 (Recipe 없음) 표시

UI Low
01

Precondition

  1. 서버에 Recipe가 0개인 상태 (초기 설치 또는 전체 삭제 후)
02

Test Steps

  1. Recipe Manager 진입
  2. 좌측 트리와 중앙/우측 상세 영역의 표시 상태 확인
  3. 안내 메시지 및 다음 액션 유도 확인
03

Test Data

Recipe count: 0
04

Expected Result

  1. 좌측 트리에 "등록된 Recipe가 없습니다" 등의 안내 텍스트
  2. 중앙/우측 영역도 Empty 상태로 깔끔히 표시 (빈 테이블 아님)
  3. "Recipe 수집 방법 안내" 또는 "Direct Upload 바로가기" 같은 다음 액션 유도
  4. 화면이 깨지거나 에러로 인식되지 않도록 시각적으로 명확히 Empty 상태임을 표현
05

비고 (QA · 개발 메모)

초기 배포 직후 또는 데모 환경에서 주로 노출될 상태.

06

제안 사항

UX 개선 Empty State에서 "장비에서 Recipe 수집 대기 중입니다" 혹은 "Direct Upload로 Recipe를 추가해보세요"와 같은 온보딩 안내 + 관련 액션 버튼을 노출하면 첫 사용자 경험이 크게 좋아집니다.

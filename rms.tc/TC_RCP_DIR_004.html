TC_RCP_DIR_004

업로드한 Recipe가 Recipe Manager에 정상 반영

Functional Critical
01

Precondition

  1. test_recipe.psr을 방금 업로드 완료
02

Test Steps

  1. Recipe Manager 진입
  2. 업로드한 Recipe가 트리에 표시되는지 확인
  3. Recipe 선택 시 Contents 영역에 파라미터 표시 확인
  4. 일반 Recipe와 동일하게 Compare, Deploy 가능한지 확인
03

Test Data

Uploaded: test_recipe.psr
04

Expected Result

  1. 업로드 즉시 (또는 새로고침 후) Recipe Manager에 표시
  2. 파라미터, 메타 정보 정상 표시
  3. 일반 Recipe와 동일하게 모든 기능 사용 가능
  4. Collected Equipment 필드는 "N/A" 또는 "Direct Upload"로 표시
05

비고 (QA · 개발 메모)

Collected Equipment: Direct Upload Recipe는 장비에서 수집되지 않았으므로 해당 필드가 빈 값 또는 "Direct Upload"로 표시되어야 함.

06

제안 사항

정보 구조 Recipe Manager 트리에 Direct Upload 아이콘(예: ⬆️)을 Custom XML 아이콘과 함께 제공하여 출처를 한눈에 파악 가능하게 해주세요.

TC_SYS_PND_003

Pending 항목 선택 후 서버 등록 실행

Functional Critical
01

Precondition

  1. Pending 항목 존재
02

Test Steps

  1. Pending 목록에서 등록할 항목 체크
  2. "Register" 또는 "서버에 등록" 버튼 클릭
  3. 진행 및 결과 확인
  4. 해당 Recipe가 Recipe Manager에 정상 등록되었는지 확인
03

Test Data

선택: EQ_001/Recipe_A
04

Expected Result

  1. 선택한 Pending 항목이 서버에 등록
  2. Pending 목록에서 해당 항목 제거
  3. Recipe Manager에 새 Recipe로 표시
  4. 등록 이력 기록
05

비고 (QA · 개발 메모)

수정 유형 처리: "수정" 유형은 기존 Recipe의 새 버전으로 등록되는지 확인.

06

제안 사항

UX 개선 "전체 등록" 원클릭 버튼 제공 시, Pending 항목이 많을 때 일괄 처리가 편리합니다.

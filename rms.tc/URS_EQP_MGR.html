URS_EQP_MGR

장비 조회 · 관리

User Requirement Core Function
01

기능 개요

RMS에 등록된 모든 장비를 한 화면에서 조회하고, 각 장비의 식별 정보와 연결 상태를 확인하는 화면이다. 사용자는 이 화면을 통해 어떤 장비가 등록되어 있고, 현재 RMS 서버와 연결된 상태인지를 파악한다.

Equipment Manager는 Recipe·Method 운영의 출발점이 되는 화면이다. 배포 대상 장비 선정과 그룹 단위 작업의 기준이 모두 이 화면의 데이터에서 시작된다.

02

사용자 니즈

주요 사용자

  • 장비 엔지니어 — 어떤 장비가 RMS에 연결되어 있는지 일상 점검
  • 공정 엔지니어 — 작업 대상 장비의 현재 상태 확인
  • RMS 관리자 — 전체 장비 자산 관리

Pain Point (RMS 도입 전)

  • 장비별 IP·ID 등 정보가 개별 문서·메모로 흩어져 있음
  • 장비가 현재 동작 중인지 한눈에 보이지 않음
  • 장비 수가 늘어날수록 관리 부담이 증가

기대 효과

  • 전체 장비 자산을 단일 화면에서 조감 가능
  • 실시간 연결 상태로 운영 가시성 확보
  • 그룹 트리를 통해 다수 장비를 체계적으로 관리
03

기능 요구사항

■ FR-1. 장비 목록 조회

  • FR-1.1 화면 진입 시 RMS에 등록된 모든 장비가 목록 형태로 표시되어야 한다.
  • FR-1.2 좌측 영역에 그룹 트리, 중앙에 장비 목록 테이블, 우측에 선택한 장비의 상세 영역이 표시된다.
  • FR-1.3 각 장비 행에는 장비 ID, 모델, IP, 그룹, 연결 상태, 마지막 통신 시각이 표시되어야 한다.

■ FR-2. 장비 상세 정보 표시

  • FR-2.1 장비를 선택하면 우측 상세 영역에 해당 장비의 모든 식별 정보가 표시된다.
  • FR-2.2 상세 정보에는 장비 ID, 모델, IP, 등록일, 등록자, 소속 그룹, Agent 버전이 포함된다.
  • FR-2.3 상세 영역에서 해당 장비에 현재 배포된 Recipe·Method 목록을 확인할 수 있다.

■ FR-3. 정렬 · 필터 · 검색

  • FR-3.1 사용자는 컬럼 헤더 클릭으로 장비 목록을 정렬할 수 있어야 한다.
  • FR-3.2 사용자는 장비 ID, IP, 모델 등으로 검색할 수 있어야 한다.
  • FR-3.3 사용자는 연결 상태(Online/Offline)로 필터링할 수 있어야 한다.
  • FR-3.4 좌측 그룹 트리에서 그룹을 선택하면 해당 그룹 소속 장비만 표시된다.

■ FR-4. 그룹 트리 표시

  • FR-4.1 좌측 그룹 트리는 RMS에 정의된 모든 장비 그룹을 계층 구조로 표시한다.
  • FR-4.2 각 그룹 노드에는 소속 장비 수가 함께 표시된다.
  • FR-4.3 그룹 트리는 펼침/접힘이 가능하며, 마지막 펼침 상태가 화면 재진입 시 유지된다.

■ FR-5. 실시간 연결 상태 표시

  • FR-5.1 각 장비의 연결 상태(Online / Offline)가 시각 아이콘으로 구분 표시되어야 한다.
  • FR-5.2 장비의 연결 상태 변경은 일정 주기 또는 이벤트 기반으로 화면에 반영된다.
  • FR-5.3 장비의 마지막 통신 시각이 함께 표시되어 사용자가 신선도를 판단할 수 있다.
04

비기능 요구사항

■ NFR-1. 성능

  • NFR-1.1 장비 100대 이하 환경에서 화면 진입 후 3초 이내에 모든 장비 목록이 표시되어야 한다.
  • NFR-1.2 장비 1,000대 환경에서도 5초 이내 초기 로딩이 가능해야 한다.
  • NFR-1.3 검색·필터 응답은 300ms 이내.

■ NFR-2. 권한

  • NFR-2.1 모든 사용자(All Roles)는 장비 목록을 조회할 수 있다.
  • NFR-2.2 등록·수정·삭제 액션은 별도 화면(Register Equipment, Edit Equipment Groups)에서 권한별로 제어된다.

■ NFR-3. 가시성

  • NFR-3.1 연결 상태 표시는 색상만이 아니라 아이콘 또는 텍스트 라벨을 병행한다(색각 이상자 대응).
  • NFR-3.2 Offline이 24시간 이상 지속된 장비는 별도 시각 표시(예: 경고 색상)로 강조한다.
05

제약사항 · 정책

  • 정책 장비 연결 상태 표시는 색상 + 아이콘 병행으로 한다.
  • 정책 장기 Offline(24시간 이상) 장비는 별도 강조 표시한다. 비정상 장비를 빠르게 발견하기 위함이다.
  • 정책 그룹 트리의 펼침 상태는 사용자별로 보존한다.
  • 제약 한 장비는 단일 그룹에만 소속된다. (Recipe·Method 그룹의 다중 소속 정책과 의도적으로 다르다.)
  • 제약 장비의 등록·수정·삭제는 이 화면이 아닌 Register Equipment / Edit Equipment Groups 화면에서 수행한다.

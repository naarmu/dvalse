TC_SYS_LGN_005

로그인 후 Role별 초기 진입 화면 확인

Functional High
01

Precondition

  1. 각 Role별 계정(Vendor, Admin, Engineer, Operator)이 준비됨
02

Test Steps

  1. Vendor 계정으로 로그인 → 진입 화면 확인
  2. 로그아웃 후 Admin으로 로그인
  3. Engineer, Operator 각각 동일 테스트
03

Test Data

4개 Role 각각 테스트
04

Expected Result

  1. 각 Role별로 기본 진입 화면이 Privilege 설정에 맞게 표시
  2. 상단 메뉴에서 접근 불가 영역은 숨김 또는 비활성화
  3. 상단에 현재 Role 표시 (slide-18 우측 상단 영역 참고)
05

비고 (QA · 개발 메모)

Privilege 연동: 각 Role이 접근 가능한 화면은 Vendor의 Privilege 설정에서 정의됨. 설정과 실제 화면이 일치해야 함.

06

제안 사항

기획 검토 각 Role의 기본 진입 화면을 명시적으로 정의해주세요. Operator는 Dashboard가 적합, Engineer는 Recipe Manager가 적합하다는 식의 기본값이 필요합니다.

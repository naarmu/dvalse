TC_RCP_DEP_007

배포 진행률 · 상태 표시

UI High
01

Precondition

  1. 다수 장비 배포 진행 중
02

Test Steps

  1. 배포 시작
  2. 진행률 바 또는 % 표시 확인
  3. 장비별 개별 상태 (Pending/In Progress/Success/Failed) 표시 확인
  4. 배포 완료 후 상태 전환 애니메이션 확인
03

Test Data

4대 장비 동시 배포
04

Expected Result

  1. 배포 시작 즉시 진행 UI 표시
  2. 장비별 상태 아이콘 또는 색상 구분 (회색=대기, 파랑=진행, 초록=성공, 빨강=실패)
  3. 진행률 % 또는 단계 텍스트 표시
  4. 완료 후 결과 요약 표시
05

비고 (QA · 개발 메모)

UI 세부는 slide-53(주요기능)의 배포 화면 참고.

06

제안 사항

UX 개선 각 장비의 상태를 카드형 UI로 표시하고, 문제 발생 시 해당 카드만 빨간 테두리로 강조하면 수많은 장비 중 이슈 장비를 빠르게 찾을 수 있습니다.

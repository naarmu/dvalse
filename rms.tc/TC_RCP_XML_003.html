TC_RCP_XML_003

등록된 XML을 장비로 배포

Functional Critical
01

Precondition

  1. 서버에 유효한 Custom XML이 등록된 상태
  2. Online 장비 2대 이상 존재
02

Test Steps

  1. 등록된 XML 중 배포할 항목 선택
  2. 대상 장비 선택 (단일 또는 다수)
  3. Deploy 실행
  4. 장비로 XML이 전송되는 과정 확인
  5. 배포 완료 시 Deployment Log에 기록 확인
03

Test Data

XML: custom_recipe.xml Targets: EQ_001, EQ_002
04

Expected Result

  1. 장비로 XML 파일이 정상 전송
  2. Deployment Log에 XML 배포 이력 기록 (일반 Recipe 배포와 구분 표시)
  3. 다수 장비 동시 배포 시 개별 결과 표시
  4. 배포 후 장비에서 XML 변환 과정으로 자동 연계됨 (TC_RCP_XML_004 참고)
05

비고 (QA · 개발 메모)

Deployment Log 표시: 일반 Recipe 배포와 XML 배포는 Log에서 구분 가능해야 함 (Type 컬럼 등).

06

제안 사항

정보 구조 Deployment Log에 "Type" 컬럼을 추가해 Recipe/Method/XML을 구분 표시하는 것을 권장드립니다.

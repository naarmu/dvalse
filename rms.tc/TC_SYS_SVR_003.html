TC_SYS_SVR_003

서버 연결 성공 시 메인 진입 흐름

Functional Critical
01

Precondition

  1. 정상 동작 중인 서버
02

Test Steps

  1. 유효한 서버 IP/Port 입력 후 연결
  2. Login 팝업 자동 표시 확인
  3. 로그인 후 메인 화면 진입 확인
03

Test Data

Valid Server: 192.168.1.10:8080
04

Expected Result

  1. 연결 성공 후 Login 팝업 자동 전환
  2. 서버 정보가 다음 실행 시 자동 적용
  3. 별도 "Server Setup" 팝업 미표시 (기 저장된 경우)
05

비고 (QA · 개발 메모)

06

제안 사항

UX 개선 저장된 서버 정보는 Settings에서 변경 가능해야 하며, 현재 연결 상태를 표시해주면 좋습니다.

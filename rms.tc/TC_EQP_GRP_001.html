TC_EQP_GRP_001

신규 장비 그룹 생성

Functional High
01

Precondition

  1. Vendor 또는 Admin 계정으로 로그인됨
  2. Edit Equipment Groups 화면 진입 완료
  3. 기존 그룹 목록 확인됨 (예: GROUP_A, GROUP_B)
02

Test Steps

  1. Add Group 또는 New Group 버튼 클릭
  2. Group Name 필드에 GROUP_NEW 입력
  3. (선택) Description 필드 입력
  4. 초기 소속 장비 선택 (선택 사항)
  5. Save 또는 Create 버튼 클릭
  6. 그룹 목록에 GROUP_NEW가 추가됨을 확인
03

Test Data

Group Name: GROUP_NEW Description: QA 테스트용 그룹 Member Equipments: EQ_001, EQ_002 (선택적)
04

Expected Result

  1. 그룹이 정상 생성되어 목록 최상단 또는 알파벳 순서 위치에 표시됨
  2. 초기에 장비를 선택했다면 해당 장비들의 그룹 속성이 GROUP_NEW로 업데이트됨
  3. Equipment Manager의 트리에서도 새 그룹이 표시됨
  4. Dashboard의 "By Equipment Groups" 필터에도 새 그룹이 추가됨
  5. 생성 이력이 Server Log에 기록됨 (생성자, 시각, 초기 멤버 수)
05

비고 (QA · 개발 메모)

그룹명 규칙: 공백·특수문자 허용 여부, 최대 길이 등 명명 규칙 확인 필요.

초기 장비 선택: 그룹 생성 시 소속 장비를 바로 지정할 수 있는지, 아니면 생성 후 편집으로만 가능한지 UX 흐름 확인.

06

제안 사항

UX 개선 Park Systems RMS Edit Equipment Groups 화면을 보면 "Display on Dashboard" 체크박스가 있습니다. 신규 그룹 생성 시 기본값이 "Dashboard에 표시 함"인지 "표시 안 함"인지 명확히 정의가 필요합니다. 기본값: 표시 함을 권장드립니다.

기획 검토 Description 필드가 실무에서 얼마나 활용되는지 확인 필요합니다. 활용도가 낮다면 제거하여 폼을 단순화할 수 있습니다.

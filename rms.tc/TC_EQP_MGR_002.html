TC_EQP_MGR_002

장비 상세 정보(IP · ID · 상태) 표시

Functional High
01

Precondition

  1. 로그인된 상태에서 Equipment Manager 화면 진입 완료
  2. 장비 EQ_001(Online) 및 EQ_002(Offline)가 서버에 등록되어 있음
  3. 각 장비에 IP, Equipment ID, 등록일, 그룹 정보가 사전 입력되어 있음
02

Test Steps

  1. 좌측 장비 트리에서 EQ_001 선택
  2. 우측 상세 영역에 표시되는 정보 항목 확인
  3. EQ_002 선택 후 동일하게 상세 정보 확인
  4. Online/Offline 상태가 상세 영역에서도 올바르게 표시되는지 확인
03

Test Data

EQ_001: IP=192.168.1.101, Group=GROUP_A, Status=Online EQ_002: IP=192.168.1.102, Group=GROUP_A, Status=Offline
04

Expected Result

  1. 장비 선택 시 우측 상세 영역에 다음 항목이 표시됨:
    • Equipment ID, Equipment Name
    • IP Address, 소속 Group
    • Connection Status (Online/Offline)
    • Agent Version, XEA Version, XEService Version
    • Deployed Recipe ID/Version, Last Updated
  2. Offline 장비는 버전 정보 등이 빈 값으로 표시되거나 "—" 등 플레이스홀더로 표시됨
  3. Online 장비는 실시간 데이터가 채워져 표시됨
  4. 표시 항목 레이블과 값이 정렬(Aligned)되어 가독성을 해치지 않음
05

비고 (QA · 개발 메모)

표시 항목 확인: RMS Dashboard 슬라이드(slide-22)의 우측 장비 카드에 나타난 항목과 Equipment Manager 상세 영역의 항목이 일관되는지 교차 확인 필요.

Offline 시점 확인: 장비가 방금 Offline이 된 경우, 이전에 수집된 버전 정보를 유지해 보여줄지, 즉시 초기화할지 정책 확인 필요.

06

제안 사항

UX 개선 Offline 장비의 상세 정보가 빈 값으로 보이면 사용자가 "데이터 로딩 실패"로 오해할 수 있습니다. "장비가 오프라인 상태여서 일부 정보를 표시할 수 없습니다" 같은 안내 문구를 상세 영역 상단에 노출하는 것을 제안드립니다.

정보 구조 현재 상세 항목이 10개 내외로 많은 편인데, 시스템 정보 / 연결 정보 / Recipe 정보 세 그룹으로 구분하면 정보 탐색 속도가 빨라집니다.

TC_MTD_GRP_005

그룹 삭제 (소속 Method 처리 방식 확인)

Functional High
01

Precondition

  1. "WAFER_METHODS" 그룹에 Method 3개 소속
02

Test Steps

  1. WAFER_METHODS 그룹 선택
  2. Delete 버튼 클릭
  3. 경고 팝업 내용 확인
  4. "확인" 선택
  5. 소속 Method들이 어떻게 되었는지 확인
03

Test Data

Group with 3 methods
04

Expected Result

  1. 삭제 시 소속 Method가 있음을 경고
  2. 확인 시 그룹만 삭제, Method는 "미지정" 상태로 복귀
  3. Method 자체는 삭제되지 않음
  4. 이력 기록
05

비고 (QA · 개발 메모)

Recipe 그룹 삭제(TC_RCP_GRP_005)와 동일 정책 적용.

06

제안 사항

일관성 Recipe 그룹과 Method 그룹의 삭제 정책을 통일해야 합니다. Equipment 그룹과도 통일을 권장드립니다.

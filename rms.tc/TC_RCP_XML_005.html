TC_RCP_XML_005

변환된 .psr이 RMS에서 조회 가능

Functional Critical
01

Precondition

  1. XML 배포 및 .psr 변환이 장비에서 완료됨
  2. Agent가 변환된 .psr을 서버로 수집한 상태
02

Test Steps

  1. Recipe Manager 진입
  2. 새로 수집된 Recipe 확인
  3. Recipe 파라미터가 원본 XML의 내용과 일치하는지 확인
  4. Description 등 메타 정보에 "Custom XML로 생성" 이력이 표시되는지 확인
03

Test Data

Expected Recipe: custom_recipe (파라미터는 XML 정의와 동일)
04

Expected Result

  1. Recipe Manager에 신규 Recipe로 표시
  2. Recipe Contents가 원본 XML의 파라미터와 일치
  3. 출처가 "Custom XML 배포"임이 Description 또는 별도 필드로 표시
  4. 일반 Recipe처럼 Compare, Deploy 등 모든 기능 사용 가능
05

비고 (QA · 개발 메모)

구분 표시: Custom XML 출신 Recipe는 일반 수집 Recipe와 시각적으로 구분되어야 운영 관리에 유리.

06

제안 사항

정보 구조 Recipe Manager 트리에서 Custom XML 출신 Recipe에 아이콘(예: 🔧)을 표시하는 것을 권장드립니다. 출처를 한눈에 파악하게 함으로써 관리가 편해집니다.

TC_SYS_LGN_002

잘못된 ID · PW 입력 시 에러 메시지

Negative High
01

Precondition

  1. Login 팝업 표시됨
02

Test Steps

  1. 존재하지 않는 ID 또는 잘못된 PW 입력
  2. Login 클릭
  3. 에러 메시지 및 상태 확인
03

Test Data

Case 1: invalid_id / any_password Case 2: valid_id / wrong_password
04

Expected Result

  1. 명확한 에러 메시지 표시 ("ID 또는 비밀번호가 일치하지 않습니다")
  2. ID와 PW 중 어느 쪽이 틀렸는지 구분해서 알려주지 않음 (보안)
  3. 실패 이력 기록 (브루트포스 대비)
  4. 입력 필드 초기화 정책 확인 (PW만 초기화 권장)
05

비고 (QA · 개발 메모)

보안: "ID가 존재하지 않습니다" 같은 구분 메시지는 사용자 열거 공격(user enumeration)에 악용됨.

06

제안 사항

보안 연속 5회 이상 실패 시 일정 시간 계정 잠금 또는 CAPTCHA 요구를 권장드립니다.

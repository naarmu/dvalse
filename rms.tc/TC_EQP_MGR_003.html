TC_EQP_MGR_003

장비 목록 정렬 (컬럼 클릭) 동작

UI Medium
01

Precondition

  1. Equipment Manager 화면 진입 완료
  2. 최소 5대 이상의 장비가 서로 다른 이름/등록일로 존재함
02

Test Steps

  1. 장비 목록 영역에서 Equipment Name 컬럼 헤더 클릭
  2. 오름차순 정렬 여부 확인
  3. 같은 컬럼 헤더 재클릭 → 내림차순 전환 확인
  4. Equipment ID, Updated Date 컬럼에 대해서도 동일 수행
  5. 정렬 상태에서 장비 트리 선택을 변경했을 때 정렬 유지 여부 확인
03

Test Data

Equipments: EQ_001 ~ EQ_005 (각기 다른 Name, Registered Date)
04

Expected Result

  1. 헤더 클릭 시 해당 컬럼 기준으로 오름차순 정렬
  2. 재클릭 시 내림차순 전환, 3번째 클릭 시 정렬 해제(기본 순서 복귀) 또는 오름차순 유지 — 정책에 따름
  3. 정렬 방향 아이콘(▲/▼)이 헤더에 명확히 표시됨
  4. 정렬 중에도 선택 상태(Selected Row)는 유지됨
  5. 장비 트리에서 다른 장비 선택 시 정렬 상태가 초기화되지 않음
05

비고 (QA · 개발 메모)

DevExpress GridControl: 기본 동작은 오름차순 → 내림차순 → 해제 3단계 토글. 3단계 동작이 정책과 맞는지 확인 필요.

Multi-column 정렬: Shift+클릭으로 다중 컬럼 정렬이 활성화되는지 여부도 같이 확인.

06

제안 사항

UX 개선 정렬 상태가 세션/계정별로 마지막 상태 기억(persist)되는지 정의가 필요합니다. RMS처럼 자주 여는 화면은 매번 같은 정렬로 여는 게 UX에 좋습니다.

기획 검토 정렬 해제(3단계) 동작을 없애고 오름차순 ↔ 내림차순 2단계 토글로 단순화하는 것도 선택지입니다. 정렬 해제가 필요한 사용자 시나리오가 실제로 있는지 검토 권장.

TC_MTD_PRM_005

설정 초기화 (기본값 복원)

Functional Medium
01

Precondition

  1. 여러 파라미터의 표시 설정이 커스텀 상태
02

Test Steps

  1. "Reset to Default" 버튼 클릭
  2. 확인 팝업 확인 후 진행
  3. 모든 파라미터가 기본 표시 상태로 복원되는지 확인
03

Test Data

커스텀 Hide 3개 → 모두 Show 복원
04

Expected Result

  1. 확인 팝업 후 모든 파라미터 기본값 복원
  2. 기본값 = "전체 표시"로 가정
  3. 초기화 이력 기록
05

비고 (QA · 개발 메모)

기본값 정의: "기본 표시 상태"가 무엇인지 명확히 정의 필요. "전체 표시"가 합리적 기본값.

06

제안 사항

UX 개선 Reset은 위험한 작업이므로 현재 설정을 Export해서 백업할 수 있는 옵션을 제안하고, 이후 Reset 진행 권장.

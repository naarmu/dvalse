TC_MTD_PRM_002

파라미터 표시 · 숨김 토글 저장

Functional High
01

Precondition

  1. Manage Method Parameters에서 파라미터 목록 표시됨
02

Test Steps

  1. "Setpoint" 파라미터의 표시 토글을 OFF로 변경
  2. 저장 버튼 클릭
  3. 저장 성공 메시지 확인
  4. 재진입 후에도 설정이 유지되는지 확인
03

Test Data

Toggle: Setpoint (Show → Hide)
04

Expected Result

  1. 토글 변경이 즉시 UI에 반영
  2. 저장 후 서버에 설정 영구화
  3. 화면 재진입 시 설정 유지
  4. 변경 이력 기록
05

비고 (QA · 개발 메모)

설정 범위: 이 설정이 전역(모든 사용자 공유)인지, 개인별인지 명확히 정의 필요. 공정 표준 관리 관점에서는 전역이 권장됨.

06

제안 사항

기획 검토 파라미터 표시 설정의 적용 범위를 확정해주세요. 권장: 전역 설정(Vendor/Admin만 변경 가능). Recipe Parameter와 동일 정책.

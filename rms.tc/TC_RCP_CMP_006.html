TC_RCP_CMP_006

Manage Recipe Parameters 설정이 비교 화면에 반영

Functional High
01

Precondition

  1. Manage Recipe Parameters에서 "Tip ID" 파라미터가 숨김 처리됨
  2. Compare Recipes 화면 진입
02

Test Steps

  1. Recipe 2개 선택하여 비교 결과 확인
  2. 파라미터 목록에서 "Tip ID" 행이 표시되지 않는지 확인
  3. Manage Recipe Parameters로 이동하여 "Tip ID" 표시 ON으로 변경
  4. Compare Recipes로 복귀하여 "Tip ID"가 비교 테이블에 나타나는지 확인
03

Test Data

Hidden parameter: Tip ID (Manage Recipe Parameters에서 off)
04

Expected Result

  1. 숨김 설정된 파라미터는 Compare Recipes 테이블에서도 숨김
  2. 표시 ON으로 변경 후 즉시(또는 재진입 시) 반영
  3. 숨김 처리된 파라미터가 비교 결과 개수(통계)에도 제외되는지 정책에 따라 확인
05

비고 (QA · 개발 메모)

적용 시점: 설정 변경 후 비교 화면에 즉시 반영되는지, 아니면 재진입 필요한지 확인. 즉시 반영이 UX에 좋음.

통계 영향: 숨김 파라미터가 "차이 개수" 등의 통계 표시에 포함되는지 여부를 기획에서 명시해야 함.

06

제안 사항

정보 구조 Compare Recipes에 "숨겨진 파라미터 N개 있음" 배너를 추가해 사용자가 숨김 상태를 인지할 수 있게 하는 것을 제안드립니다. 공정 검토 시 "왜 이 파라미터는 안 보이지?"로 혼란이 발생할 수 있습니다.

기획 검토 숨김 처리된 파라미터를 "일시적으로 보기" 토글로 임시 해제할 수 있으면 편리합니다. Manage Recipe Parameters로 왕복하지 않아도 됩니다.

TC_SYS_SVR_005

Server Setup 설정값 재접속 시 유지

Functional High
01

Precondition

  1. 이전에 Server Setup 완료 및 정상 로그인 이력 있음
02

Test Steps

  1. Client 종료
  2. Client 재실행
  3. Server Setup 팝업이 다시 뜨지 않는지 확인
  4. Login 팝업이 바로 표시되는지 확인
03

Test Data

이전 설정: 192.168.1.10:8080
04

Expected Result

  1. Client 재실행 시 Server Setup 팝업 스킵
  2. Login 팝업으로 바로 진입
  3. 이전 저장된 서버 정보로 자동 연결
05

비고 (QA · 개발 메모)

저장 위치: Windows 레지스트리 또는 AppData 폴더에 저장될 것으로 추정. 저장 위치 확인 필요.

06

제안 사항

보안 저장된 서버 정보는 일반 사용자가 수정 불가한 위치에 저장하는 것이 좋습니다. 악의적 수정으로 피싱 서버 연결 방지.

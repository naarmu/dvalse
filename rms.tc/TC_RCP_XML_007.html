TC_RCP_XML_007

다수 장비에 동일 XML 동시 배포

Functional High
01

Precondition

  1. 유효한 Custom XML 서버 등록됨
  2. Online 장비 3대 이상
02

Test Steps

  1. 등록된 XML 선택
  2. 대상 장비 3대 이상 선택
  3. Deploy 실행
  4. 각 장비에서 변환이 병렬로 진행되는지 확인
  5. 모든 장비에서 동일한 .psr이 생성되는지 확인
03

Test Data

XML: custom_recipe.xml Targets: EQ_001, EQ_002, EQ_003
04

Expected Result

  1. 3대 장비에 동시 배포 성공
  2. 각 장비에서 변환이 병렬 수행
  3. 생성된 .psr이 서버로 수집되면 동일 파라미터를 가진 Recipe로 등록 (장비별 동일 Recipe 존재)
  4. Deployment Log에 장비별 개별 이력 기록
05

비고 (QA · 개발 메모)

Recipe 중복: 동일 XML을 여러 장비에 배포 시 RMS에 같은 이름의 Recipe가 여러 개 등록될 수 있음. 정책 확정 필요.

06

제안 사항

기획 검토 다수 장비에 동일 XML 배포 시 RMS Recipe 관리 정책이 필요합니다. 대안: ① 장비별 Recipe를 각각 등록, ② 하나의 Recipe로 통합 관리하고 여러 장비에 배포 이력만 기록. 후자가 관리 편의성이 높습니다.
